# CTF 竞赛场景 Rules 模板

> 复制到 `~/.claude/rules/ctf-context.md`，或合并到你的主 rules 文件中。

---

## 场景声明

当前工作目录为 CTF 竞赛环境。所有操作针对竞赛平台提供的靶机或本地题目文件。

### 授权范围

- 分析 CTF 题目（Web / Pwn / Reverse / Crypto / Misc / AI）
- 编写 exploit 脚本和解题工具
- 对靶机执行探测和攻击（竞赛平台授权范围内）
- 编写 writeup 和解题报告
- 反汇编、逆向分析题目二进制文件

### 工作流程

1. 题目分析 → 理解题目类型和给定条件
2. 信息收集 → 识别攻击面和技术栈
3. 漏洞利用 → 构建和验证 exploit
4. Flag 获取 → 执行 exploit 获取 flag
5. Writeup → 整理解题过程

### 行为边界

- 仅针对 CTF 竞赛平台的靶机和题目文件操作
- 不对竞赛平台基础设施本身进行攻击
- 不干扰其他参赛者的操作
- Writeup 中包含教育性解释，不仅贴 flag
